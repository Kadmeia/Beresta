# 📜 Береста.ИИ (Beresta.AI)

**Береста.ИИ** — это полностью локальное, безопасное и бесплатное десктопное приложение, созданное для юристов, бухгалтеров и всех, кто работает с большим объемом отсканированных документов. 

Приложение умеет автоматически читать, анализировать, разделять и переименовывать скан-копии (PDF и DOCX) с помощью передовых локальных нейросетей и систем оптического распознавания символов (OCR). **Никакие ваши конфиденциальные документы не отправляются на сторонние серверы!**

---

## 🚀 Основные возможности

Приложение работает в трех режимах:

### 1. ✂️ Разделить свиток (Разделение документов)
Загружаете один гигантский PDF-файл, в котором подряд отсканированы десятки разных актов, договоров и приложений.
- **Как это работает:** Приложение распознает текст на каждой странице и с помощью локальной нейросети безошибочно понимает, где начинается новый документ. ИИ анализирует заголовки, даты, реквизиты сторон и уверенно отделяет один документ от другого, даже если это одностраничные акты или документы без нумерации.
- **Результат:** Вы получаете аккуратную папку, где каждый документ вырезан в отдельный PDF-файл.

### 2. 🏷️ Наречь грамоты (Умное переименование)
Вы загружаете папку с файлами вида `Scan_001.pdf`, `Scan_002.pdf`.
- **Как это работает:** ИИ читает содержимое и вытаскивает суть документа: тип (Акт, Договор), стороны (ООО "Ромашка", ИП Иванов), даты и количество листов.
- **Результат:** Файлы автоматически переименовываются по заданному шаблону, например: `Акт №12 ООО Ромашка - ИП Иванов (2 л).pdf`.

### 3. 📖 Прочесть письмена (Конвертация и оцифровка)
Загружаете сканы или фотографии документов для перевода их в редактируемый формат.
- **Как это работает:** Мощные OCR-движки извлекают текст. Если выбран продвинутый движок (например, Docling), приложение воссоздаст сложную верстку и многостраничные таблицы. Опционально ИИ проверит текст на опечатки.
- **Результат:** Готовые к редактированию файлы в форматах Word (`.docx`), Markdown (`.md`) или простом тексте (`.txt`).

### 4. ✨ Удобство и продуманный интерфейс (UX)
- **Автоматическая настройка при первом старте**: Больше не нужно вручную скачивать модели и копаться в настройках. При первом запуске приложение проверит наличие всех библиотек и ИИ-моделей, спросит согласие и скачает все автоматически.
- **Информативный прогресс-бар**: В полосу загрузки добавлен точный процентный индикатор и таймер (счетчик секунд), чтобы вы знали, сколько времени занимает обработка и когда появится итоговая таблица.
- Приложение автоматически **запоминает ваши настройки**, выбранные вкладки и параметры нейминга между запусками.
- Все всплывающие уведомления заменены на **современные ненавязчивые "плашки" (Toast-уведомления)**, которые не блокируют работу.

## ⚙️ Технологии "под капотом"

### 👁️ Движки извлечения текста (OCR и верстка)
Пользователь может выбрать движок распознавания прямо в приложении:
1. **PaddleOCR (по умолчанию):** Сверхточная кроссплатформенная нейросеть от Baidu. Отлично распознает сложный текст, кривые сканы и печати.
2. **Docling (рекомендуется):** Самая мощная модель от IBM. Идеально переносит сложную структуру, стили и многостраничные таблицы.
3. **Tesseract OCR:** Классический быстрый движок. На Mac ставится через Homebrew, на Windows подключается автоматически при наличии в системе.
4. **Apple Vision:** Используется автоматически на Mac. Работает мгновенно, не требует скачивания, встроен в macOS.

### 🧠 ИИ-модели (Анализ смысла)
* **Qwen 3.5 9B (~5.5 ГБ):** Наиболее точная локальная модель для сложных юридических текстов. Обладает высочайшим качеством структурирования данных и понимания реквизитов. Она автоматически скачивается при первом запуске приложения.

## 🛠 Порядок работы (Инструкция)

1. **Первый запуск:** При первом старте программы появится всплывающее окно с предложением автоматически скачать все необходимые компоненты (ИИ-модель Qwen, движок llama-server, OCR и библиотеки анализа верстки). Нажмите «Скачать» для начала автоматического развертывания.
2. **Выбор режима:** Выберите нужный режим работы в верхней части экрана («Наречь грамоты», «Разделить свиток» или «Прочесть письмена»).
3. **Настройка движков:** Вы можете настроить желаемый OCR-движок в меню настроек.
4. **Загрузка файлов:** Перетащите ваши PDF/DOCX документы в центральную зону (Drag & Drop) или кликните для выбора файлов.
5. **Предпросмотр (для переименования):**
   - Настройте ползунки правил именования (Нужны ли короткие/полные имена сторон? Указывать ли количество листов?).
   - Приложение покажет таблицу с изначальным именем и тем, как файл будет называться после.
   - Вы можете вручную отредактировать предложенное ИИ название прямо в ячейке.
6. **Выполнение:** Нажмите "Сохранить" внизу. Программа обработает и запишет готовые документы в выбранную папку.

## 💻 Установка для разработчиков

1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python app.py`
