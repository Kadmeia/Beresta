# 📜 Береста.ИИ (Beresta.AI)

**Береста.ИИ** — это полностью локальное, безопасное и бесплатное десктопное приложение, созданное для юристов, бухгалтеров и всех, кто работает с большим объемом отсканированных документов. 

Приложение умеет автоматически читать, анализировать, разделять и переименовывать скан-копии (PDF и DOCX) с помощью передовых локальных нейросетей и систем оптического распознавания символов (OCR). **Никакие ваши конфиденциальные документы не отправляются на сторонние серверы!**

---

## 🚀 Основные возможности

Приложение работает в двух режимах:

### 1. ✂️ Разделить свиток (Разделение документов)
Загружаете один гигантский PDF-файл, в котором подряд отсканированы десятки разных актов, договоров и приложений.
- **Как это работает:** Приложение распознает текст на каждой странице и с помощью нейросети понимает, где заканчивается один документ и начинается другой (даже если у них нет четкой нумерации).
- **Результат:** Вы получаете аккуратную папку, где каждый документ вырезан в отдельный PDF-файл.

### 2. 🏷️ Наречь грамоты (Умное переименование)
Вы загружаете папку с файлами вида `Scan_001.pdf`, `Scan_002.pdf`.
- **Как это работает:** ИИ читает содержимое и вытаскивает суть документа: тип (Акт, Договор), стороны (ООО "Ромашка", ИП Иванов), даты и количество листов.
- **Результат:** Файлы автоматически переименовываются по заданному шаблону, например: `Акт №12 ООО Ромашка - ИП Иванов (2 л).pdf`.

## ⚙️ Технологии "под капотом"

### 👁️ OCR-движки (Распознавание текста)
Пользователь может выбрать движок распознавания прямо в приложении:
1. **Apple Vision:** Используется автоматически на Mac. Работает мгновенно, не требует скачивания, встроен в macOS.
2. **Tesseract OCR:** Классический быстрый движок. На Windows скачивается портативной сборкой в 1 клик прямо из приложения.
3. **PaddleOCR:** Сверхточная кроссплатформенная нейросеть от Baidu. Отлично распознает сложный текст и печати. Устанавливается по нажатию кнопки.

### 🧠 ИИ-модели (Анализ смысла)
1. **Локальная нейросеть (Qwen Saiga):** Специально обученная русскоязычная модель, которая скачивается и работает прямо на вашей видеокарте/процессоре. Обеспечивает 100% приватность (интернет не нужен).
2. **Облачные API:** Если локальной мощности не хватает, можно подключить ключи от OpenAI (ChatGPT), Anthropic (Claude), Google (Gemini) или DeepSeek.

## 🛠 Порядок работы (Инструкция)

1. **Запуск:** Откройте приложение. Выберите нужный режим работы в верхней части экрана («Разделить свиток» или «Наречь грамоты»).
2. **Настройка движков:**
   - Нажмите на кнопку **«OCR-движок»** в правом верхнем углу и выберите систему распознавания.
   - Нажмите на **«ИИ-модель»** и выберите, кто будет "думать" над вашими документами (локальная модель или API).
3. **Загрузка файлов:** Перетащите ваши PDF/DOCX документы в центральную зону (Drag & Drop) или кликните для выбора файлов.
4. **Предпросмотр (для переименования):**
   - Настройте ползунки правил именования (Нужны ли короткие/полные имена сторон? Указывать ли количество листов?).
   - Приложение покажет таблицу с изначальным именем и тем, как файл будет называться после.
   - Вы можете вручную отредактировать предложенное ИИ название прямо в ячейке.
5. **Выполнение:** Нажмите большую зеленую кнопку внизу. Программа создаст новую папку и положит туда готовые, красивые документы.

## 💻 Установка для разработчиков

1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python app.py`

*Создано с душой от автора DOCXдодыра. Поддержать проект можно по кнопке в приложении.*
